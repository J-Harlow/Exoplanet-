This project presents an enriched catalog of 6,059 confirmed exoplanets and their host stars through a multi-stage data processing pipeline built in Python and Jupyter Notebook. The workflow integrates information from the NASA Exoplanet Archive PSComppars table, the SIMBAD Astronomical Database, and derived stellar classification methods to improve dataset completeness. Missing planetary and stellar parameters—including orbital period, orbital distance, planetary radius, mass, equilibrium temperature, stellar temperature, radius, gravity, and spectral type—were recovered using automated database queries and rule-based astrophysical estimation.

The final dataset supports large-scale exoplanet population analysis through statistical visualization and host-star characterization. Included notebooks perform NASA enrichment, SIMBAD matching through Astroquery, stellar spectral derivation, and graph generation for publication-ready analysis. The resulting catalog enables investigation of planetary distributions, mass–radius relationships, orbital architecture, spectral populations, and H-R diagram placement, while preserving reproducibility through a transparent, code-driven workflow.
